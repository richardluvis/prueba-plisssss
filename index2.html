Este proyecto es una aplicación web educativa desarrollada como parte de un proyecto universitario para la universidad de San Buenaventura sede Bogotá D.C. El objetivo principal de LEARNING 4 ALL es llevar educación gratuita y de calidad a todas las personas interesadas en los estudios relacionados con las nuevas tecnologías de software.

Objetivo General

Desarrollar una plataforma web para el aprendizaje del desarrollo de software.

Pregunta Problema

¿Cómo facilitar el acceso a la educación enfocada a lenguajes de programación en Colombia?

Funcionalidades

  • Página de Inicio: Presenta una descripción general del proyecto educativo.
  • Cursos: Sección que contiene materiales educativos para los usuarios.
  • Contacto: Información de contacto del equipo de desarrollo.
  • Registro de Usuarios: Permite a los usuarios crear cuentas para acceder a contenido personalizado.
  • Inicio de Sesión: Los usuarios registrados pueden iniciar sesión para acceder a funcionalidades adicionales.
  • Recursos Interactivos: Incluye vídeos, cuestionarios y actividades interactivas para mejorar la experiencia de aprendizaje.
  • Gestión de Contenidos: Permite a los administradores del sitio actualizar y gestionar el contenido de forma sencilla.
  • Diseño Responsivo: Garantiza que la página web se vea bien y sea fácil de usar en dispositivos móviles y tablets.
  • Seguridad: Implementa medidas de seguridad para proteger los datos de los usuarios y prevenir posibles vulnerabilidades.

Instrucciones de Instalación

  1. Clonar el repositorio desde GitHub.
  2. Léase [requerimientos.txt]
  3. Instalar las dependencias utilizando npm install.
  4. Ejecutar la aplicación con node server.js.
  5. Acceder a la página web en http://localhost:4000.
